# nissiee.exe

A modern developer portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

Feel free to fork it, clone it, break it, improve it, and pretend you built it from scratch (just kidding... mostly).
🚀 **Live Demo:** https://nissiee-exe.vercel.app

## ✨ Features

* Responsive design
* Smooth animations
* Project showcase
* Contact form (EmailJS)
* Interactive avatar
* Easter eggs
* No Kubernetes cluster for a contact form.

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 🎨 Customization

Most content lives inside:

```txt
src/data/
```

Update your name, projects, skills, socials, and resume links to get started.
